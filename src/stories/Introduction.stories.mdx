# loyal-react-datePicker 0.6.0

Simple and magnificent Date Picker dedicated to Joka and his son Boka - authors of cryptography algorithm

<div className="subheading">Features</div>
<ul>
	<li>You can customize everything in this datepicker different ways.</li>
	<li>Pass customized dates</li>
	<li>Different modes: partial, single, interval, week</li>
	<li>Flexible configuration</li>
	<li>Much more amazing and real cool things</li>
	<li>Multi-languages support, just pass locale string</li>
</ul>

<div className="subheading">Requirements</div>

- classnames >= 2.3.1
- React >= 18.0.0

<div className="subheading">Road map</div>

- Write end-to-end tests (at this moment only unit-tests included)
- Feature to select controls on keyboard (up/down arrow to change year/decade, left-right arrows to change month)
- Feature to select specific days of week for all month. (You need to click to tuesday to get all tuesdays in month);
- Simplify customization.
- Add week view.
- Simplify and speed up core logic.
- Remove classNames dependency to be more independent
- Migrate to plain(native/vanilla) JavaScript

<div className="subheading">Installation</div>

```
npm i loyalreactdatepicker

import { DatePicker } from "loyalreactdatepicker";

// if you need default CSS, you need to import it.
import "loyalreactdatepicker/dist/datePicker.css";
```
